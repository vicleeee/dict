# 凡人修仙传 - 英语单词拼写游戏

一个以修仙为主题的英语单词拼写闯关游戏，支持14本教材、按单元灵活选择背诵范围。

## 演示地址

🔗 **[https://lxl.bella.press/dict-new/index.html](https://lxl.bella.press/dict-new/index.html)**

📊 教师后台：[https://lxl.bella.press/dict-new/teacher.html](https://lxl.bella.press/dict-new/teacher.html)

## 界面预览

### 欢迎页
输入道号（姓名），可选配置 QuickForm API 地址，点击「开始修仙之旅」。

### 选择修仙秘籍
小学（练气期）和初中（筑基期）共14本教材，点击任意一本进入单元选择。

### 单元选择
勾选一个或多个单元，底部实时显示"从 X 词中随机10题"，支持全选/取消全选。

### 游戏界面
显示中文释义，用键盘输入英文单词拼写。答对+10分，配有音效反馈。

### 结果页
展示得分、正确/错误单词、用时。支持生成修仙证书（截图下载），提交成绩到 QuickForm 后台。点击「再次修仙」保持当前教材和单元选择，重新随机10题。

## 功能特点

- **14本教材词库**：PEP人教版（3-6年级）+ 外研社（7-9年级）
- **单元级筛选**：可选择一本书的一个或多个单元，精准控制背诵范围
- **随机抽题**：每次从选中单词池中随机抽取10题
- **修仙主题**：练气期/筑基期分级，修仙证书截图
- **教师后台**：排行榜、学习记录查询、数据导出
- **QuickForm 集成**：成绩自动提交到 QuickForm 后台，支持自定义 API 地址
- **响应式设计**：支持电脑和手机端

## 使用方法

### 学生端

1. 打开 `index.html`
2. 输入姓名，点击「开始修仙之旅」
3. 选择一本教材
4. 勾选要背诵的单元（可多选），点击「开始修仙」
5. 看中文打英文，完成10题
6. 查看成绩，可生成证书截图

### 教师端

1. 打开 `teacher.html`
2. 查看排行榜和全部学习记录
3. 支持按学生姓名搜索
4. 点击「导出数据」下载 CSV

### 配置 QuickForm 后台

默认已配置 QuickForm API。如需使用自己的后台：

1. 在欢迎页「QuickForm API 地址」输入框中填写你的 API 地址
2. 或在教师后台顶部填写并保存
3. 两处共享同一配置（localStorage），配置一次即可

## JSON 词库结构

```json
{
  "name": "PEP三年级上册词汇表（新课标）",
  "type": "pep",
  "units": [
    {
      "name": "Unit 1 Making friends",
      "words": [
        {"英文": "nice", "中文": "令人愉快的；友好的"},
        {"英文": "ear", "中文": "耳朵"}
      ]
    }
  ]
}
```

兼容旧版扁平格式（无 `units` 字段时直接整本书抽题）。

## 部署

静态文件，托管到任意 HTTP 服务器即可：

- **腾讯云 COS**：开启静态网站托管，绑定自定义域名
- **GitHub Pages**：直接 push 后启用 Pages
- **Nginx / Apache**：复制文件到 web 目录

## 技术栈

- 纯前端 HTML/CSS/JavaScript
- 无框架依赖
- 音效：Web Audio API
- 截图：html2canvas
- 数据提交：QuickForm API
